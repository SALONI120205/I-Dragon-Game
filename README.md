🐉 I-Dragon Game

A responsive endless runner game built using HTML, CSS, and JavaScript.
The player controls a dragon that jumps over obstacles while the game speed gradually increases. The project demonstrates collision detection, animation control, state management, and sound integration.

🚀 Live Demo

🔗 https://i-dragongame.netlify.app

🎮 Features

🐉 Smooth dragon jump mechanics

⬆️ Space & Arrow key controls

📱 Mobile tap support

🎵 Background music & game over sound

💥 Collision detection with soft hitbox

📈 Progressive speed increase

🏆 High score saved using LocalStorage

🔁 Restart functionality

🎨 Custom assets and background

🛠️ Tech Stack
